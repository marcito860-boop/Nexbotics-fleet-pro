## Bulk Import/Export for Maintenance Records (NEW FEATURE)

### Overview
Added mass upload capability for maintenance records, similar to the original fleet-system.

### Features
**Import Types Supported:**
- vehicles
- drivers  
- inventory
- **maintenance_records** (NEW)

**CSV Template Columns:**
```csv
vehicle_registration,service_type,category,title,description,provider_name,
scheduled_date,completed_date,service_mileage,next_service_mileage,
labor_cost,parts_cost,other_cost,status,technician_name,invoice_number,warranty_months,notes
```

**API Endpoints:**
```bash
# Preview CSV before import
POST /api/fleet/import/preview
Body: { "importType": "maintenance_records", "csvContent": "..." }

# Start import job
POST /api/fleet/import
Body: { "importType": "maintenance_records", "csvContent": "...", "fileName": "maintenance.csv" }

# Download template
GET /api/fleet/export/templates/maintenance_records

# Export existing records
POST /api/fleet/export
Body: { "exportType": "maintenance_records", "format": "csv" }

# Check import job status
GET /api/fleet/import/jobs/:id
```

**Import Logic:**
- Looks up vehicle by `vehicle_registration` (required)
- Looks up provider by `provider_name` (optional, auto-matches if found)
- Calculates `warranty_expiry` from `warranty_months + completed_date`
- Updates vehicle mileage when record has `completed_date` and `service_mileage`
- Validates service_type: `preventive`, `repair`, `breakdown`, `emergency`

**Sample CSV Row:**
```csv
KDD 413A,preventive,Engine,Oil Change,Regular oil change service,Toyota Kenya,2025-03-15,2025-03-15,45200,50200,1500,800,200,completed,John Doe,INV-001,3,Used synthetic oil
```
