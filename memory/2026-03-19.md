### Bug Fix: JavaScript Error on VehiclesPage
User reported console error: `TypeError: Cannot read properties of undefined (reading 'replace')`

**Root cause**: In `filteredVehicles`, the code was calling `.toLowerCase()` on vehicle properties without null checks:
```javascript
// BEFORE (crashes if null):
v.registrationNumber.toLowerCase()

// AFTER (safe):
(v.registrationNumber || '').toLowerCase()
```

Also fixed:
- `vehicle.mileage.toLocaleString()` → `(vehicle.currentMileage || vehicle.mileage || 0).toLocaleString()`
- Similar fixes in DriversPage for `employmentStatus.replace('_', ' ')`
