## Training Module Enhancement

### Features Added
1. **Permanent Course Notes** - 8 comprehensive courses with reference materials
2. **15-Question Randomized Quizzes** - Generated from course notes
3. **3-Attempt Lockout System** - Max 3 attempts before admin unlock required
4. **Certificate Generation** - PDF certificates with name, credentials, company
5. **Admin Unlock Requests** - Users can request unlock after max attempts

### Course Topics
- Defensive Driving Fundamentals
- Vehicle Inspection & Maintenance
- Fuel Efficiency & Eco-Driving
- Accident Prevention & Response
- Regulatory Compliance & Hours of Service
- Cargo Securement
- Hazardous Materials Transportation
- Security & Anti-Terrorism

### Key Features

**Course Notes System:**
- Notes stored in `/backend/src/data/courseNotes.ts`
- Structured sections with key learning points
- Auto-displayed during course viewing
- Admin can update notes per course

**Quiz System:**
- Exactly 15 questions per quiz
- Randomized from question templates + generated from notes
- 30-minute time limit
- Questions shuffled for each attempt
- Immediate feedback after each answer

**Attempt Management:**
- Max 3 attempts per course
- After 3 failures, user locked out
- Can submit unlock request with reason
- Admin approves/denies requests
- Approved requests reset all attempts

**Certificate Generation:**
- Generated on passing (70%+ score typically)
- Includes: Full name, credentials, company, course, score, date
- Unique certificate number (FLT-XXXXXXXX-XXXX format)
- Downloadable PDF
- Stored in database for verification

### Database Schema Additions
- `quiz_unlock_requests` table - Tracks unlock requests
- `quiz_questions_generated` table - Stores attempt-specific questions
- Added columns: `courses.permanent_notes`, `certificates.user_credentials`, `certificates.company_name`, `certificates.pdf_data`

### API Endpoints
| Endpoint | Description |
|----------|-------------|
| `GET /courses` | List courses with notes |
| `GET /courses/:id/notes` | Get course notes only |
| `POST /courses/:id/start-quiz` | Start new quiz attempt |
| `POST /attempts/:id/answer` | Submit answer |
| `POST /attempts/:id/complete` | Finish quiz & generate certificate |
| `GET /my-certificates` | View certificates |
| `GET /certificates/:id/download` | Download PDF |
| `POST /courses/:id/request-unlock` | Request quiz unlock |
| `GET /unlock-requests` | Admin: view pending requests |
| `POST /unlock-requests/:id/approve` | Admin: approve unlock |
| `POST /unlock-requests/:id/deny` | Admin: deny unlock |
